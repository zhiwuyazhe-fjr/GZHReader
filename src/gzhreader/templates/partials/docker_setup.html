下载 Docker Desktop 查看安装说明

下一步怎么做

1. 安装 Docker Desktop
点击上面的官方下载按钮，按官方向导安装即可。
2. 启动 Docker Desktop
首次启动通常会稍慢，请等到界面显示 Engine running 或类似“已就绪”的状态。
3. 回到这里点“重新检测”
检测通过后，这里会自动恢复为正式向导，你就可以继续启动 wewe-rss-app 和 mysql 了。
为什么必须先装 Docker？
因为 GZHReader 依赖 Docker Desktop 来启动 wewe-rss-app 和 mysql。没有它，RSS 服务就没法在本机跑起来。
