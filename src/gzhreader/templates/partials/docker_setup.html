下载 Docker Desktop 查看安装说明
上面两个链接都会在新页面打开。先按官方文档完成安装，再回到这里继续。
为什么这一步值得继续做
Docker Desktop 只负责把 RSS 服务稳定地跑在你的本机里。装好之后，后面的 RSS、LLM、计划任务和日报导出都可以直接在这个界面继续完成。
官方来源 下载按钮直达 Docker 官方 Windows 安装文档。
首次安装即可复用 后续升级 GZHReader 时通常不需要重新配置 Docker。
过程中可能会弹出系统窗口 安装或后续启动服务时，如果短暂打开浏览器、终端或系统确认窗口，都属于正常行为。

三步完成首次安装

整个过程通常只需要几分钟。完成后，这个页面会恢复成正式的 8 步设置向导。
安装完成后无需留在 Docker 官网
01
打开官方 Windows 安装页
点击“下载 Docker Desktop”后会新开浏览器页面，直接跟随 Docker 官方步骤下载和安装。
02
启动 Docker Desktop 并等待引擎就绪
首次启动通常稍慢。等到 Docker Desktop 显示 Engine running 或类似“已就绪”状态即可。
03
回到 GZHReader 继续配置
点击“重新检测”后，如果 Docker 已就绪，页面会自动切回正式向导，继续启动 wewe-rss-app 和 mysql。
